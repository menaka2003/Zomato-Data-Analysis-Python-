# Zomato-Data-Analysis-Python-
Cleaned and analyzed Zomato data using Python to derive customer dining insights.
Project: Zomato Data Analysis 

 1. Project Goal
 The objective was to perform Exploratory Data Analysis (EDA) on Zomato restaurant data to identify key customer preferences, market trends, and valuable business insights.

 2. Skills & Tools
* Languages: Python (Pandas, NumPy, Matplotlib, Seaborn)
* Data Handling: Data Cleaning, Missing Value Imputation, Duplicate Removal

 3. Key Findings & Business Impact
* Successfully analyzed data achieving 90% exploration and 85% visualization for customer insights.
* Specific Insight: Identified that 40% of all listed restaurants accept online orders, indicating a major channel for business growth.
* Business Value: Generated clear visualizations of the top 10 most liked dishes and city-wise restaurant density to guide menu and location expansion strategies.
